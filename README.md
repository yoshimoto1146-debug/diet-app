# 整骨院AIダイエットコーチ

整骨院（接骨院）の患者様向けに特化した、AI搭載型ダイエット・健康管理アプリです。

## 主な機能

- **AI食事分析**: 写真やテキストから、カロリー・PFCバランス（タンパク質・脂質・炭水化物）を瞬時に解析。
- **1日の食事採点**: 1日の食事を100点満点で評価し、コーチが具体的なアドバイスを提供。
- **InBody連携**: 結果用紙をカメラで撮るだけで数値をデータ化し、グラフで推移を表示。
- **パーソナライズ提案**: 性別、年齢、仕事（デスクワーク/重労働など）、運動習慣に基づいた、整骨院ならではの運動・ストレッチメニューを提案。

## 技術スタック

- **Frontend**: React (TypeScript)
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **AI**: Google Gemini API (gemini-3-flash-preview)
- **Charts**: Recharts

## セットアップ

1. リポジトリをクローン
2. `npm install` で依存関係をインストール
3. Gemini APIキーを環境変数 `API_KEY` に設定
4. `npm run dev` で起動
